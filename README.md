# CARE

Visualization application that brings CARE's donation data to life.

## Installation

Requirements:

* NodeJs 5.2+ [How to install](https://nodejs.org/download/)

Install dependencies:

	npm install

## Usage

To run the server:

	npm start

To build:

	npm run build
